#### 实时通信 letschat

功能

1. 添加好友
2. 单聊 群聊 广播
3. 通告
4. 聊天记录存储
5. AI聊天

优化

1. 消息防抖和限流
2. 安全认证 连接时验证用户身份

技术栈

- 前端

  - UI 框架

    - React v19.1

  - css 框架

    - sass
    - tailwind css

  - 脚本语言

    - typescript

  - 测试

    - jest
    - mock

  - 运维

    - github Action CI/CD
    - docker
    - 云服务部署

  - 工程化

    -  webpack
    - bebel 
    - eslint

  - 服务端

    - node v20.13.1 npm v9.5.0
    - express

  - API 设计范式

    - restful api
  
  - 数据库 

    - mongodb 
    - Compass 工具
    - MongoDB for VS Code 插件
